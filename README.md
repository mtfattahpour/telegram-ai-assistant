# Telegram AI Assistant: Clinical & Academic  Knowledge Synthesizer

A Python-based, event-driven tool for filtering, searching, and synthesizing Telegram messages. By integrating intelligent summarization and AI-assisted knowledge extraction, this tool allows users to query massive groups or channels in natural language and receive cited answers via a dedicated chat interface.

## Project Motivation & Context

At major medical institutions in Iran, including Tehran University of Medical Sciences (TUMS) and its affiliated teaching hospitals, Telegram channels and supergroups have become de facto repositories for real-time clinical knowledge. Daily morning reports, complex case discussions, and unwritten clinical pearls are constantly shared in these spaces.

As an MD graduate, I still actively follow these channels to stay updated. However, the sheer volume of unstructured messages makes finding specific information and insights incredibly inefficient. Standard search functionality implemented by Telegram UI is often insufficient for complex queries, and it is often time-consuming to manually search for the answer to a specific question.

This project leverages the `Telethon` library and Generative AI (Google Gemini) to act as a specialized Retrieval-Augmented Generation (RAG) agent, transforming chaotic chat histories into a highly searchable, insightful clinical database. 

## Core Capabilities (v0.1 - Stable Prototype)

This project is an ongoing endeavor. The current stable prototype operates via a dedicated Telegram chat interface (automatically generated by the script), allowing users to interact with the agent natively.

- **Agentic Context-Aware Q&A (`/ask` command)**: The flagship feature. Users input a natural language clinical/research query and target a specific channel. The script:
    1. Performs iterative context fetching and pagination from the target channel.
    2. Synthesizes a grounded response based only on the retrieved discussions.
    3. Returns verifiable citations, including deep-links to the original Telegram messages for manual verification.
- **Intelligent Summarization**: Summarization of specific message chains or case discussions to prevent information overload.
- **Multilingual Processing**: Native support for English and RTL languages (specifically Persian/Farsi).

## Roadmap & Future Development

This project is under active development. Upcoming features include:

- **Advanced Search**: fuzzy string matching for typos and variations, search based on custom time range, specific sender(s), media attachments, intelligent keyword based filtering, retrieval of message chains (a message and all its replies).
- **Data Handling**:
    - Proper parsing and storage of message data (text, sender, date).
    - Future support for handling images and documents.
- **Advanced Media Handling**: Basic OCR and vision-model integration to extract text/context from shared lab results, EKGs, and radiologic reports posted in groups.
- **Caching and Performance Increase**
- **Advanced LLM Control**: Choosing models, generative AI parameters, system prompts, context window length limitation, integration of web search, etc.

## Setup & Installation

1.  **Prerequisites**: Python 3.8+
2.  **Clone the repository**:
    ```bash
    git clone https://github.com/mtfattahpour/telegram-ai-assistant.git
    cd telegram-ai-assistant
    ```
3.  **Create a virtual environment and install dependencies**:
    ```bash
    # Create the environment
    python -m venv .venv
    
    # Activate the environment (Windows)
    .venv\Scripts\activate
    # OR Activate (macOS/Linux)
    source .venv/bin/activate

    pip install -r requirements.txt
    ```
4.  **Obtain API Credentials**:
    - **Telegram**: Go to https://my.telegram.org/apps to create a `Desktop` app and obtain your `api_id` and `api_hash`.
    - **Gemini**: Obtain an API key from Google Gemini.
5.  **Configure Environment Variables**:
    - Create a file named `.env` in the root directory.
    - Add your credentials to this file:
      ```env
      API_ID=your_api_id_here
      API_HASH=your_api_hash_here
      GEMINI_API_KEY=your_gemini_api_key_here
      ```
6.  **Execution**:
    ```bash
    python main.py
    ```
    - *Note: On the first run, `main.py` will prompt you for your Telegram login code to automatically generate a secure `.session` file locally. The file grants programmatic access to Telegram account and should never be shared.*
    - Once running, the script creates a new Telegram group which acts as your UI interface. Use the `/help` command in that chat to view all available analytical functions. Send `/stop` or press `Ctrl+C` in the terminal to terminate the script.
